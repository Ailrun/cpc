Enum Mir

pub enum Mir {
    Empty,
    Loop(Box<Mir>),
    Maybe(Box<Mir>),
    Concat(Vec<Mir>),
    Alternation(Vec<Mir>),
    Class(Class),
    Literal(Literal),
}
Expand description

Middle Intermediate Representation of the regex, built from regex_syntax’s Hir. The goal here is to strip and canonicalize the tree, so that we don’t have to do transformations later on the graph, with the potential of running into looping references.
