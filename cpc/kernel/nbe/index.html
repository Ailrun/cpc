Â§Normalization Module Based on Normalization-by-Evaluation (NbE)

This module provides normalization methods (nbe and nbe_typ) for a well-typed term (and well-formed type). Note that, for well-typed term, normalization should never fail, any errorneous cases in this module call panic!.

Actual implementation lives in trait implementations of Exp. Specifically, one can check its nbe method and eval method.
