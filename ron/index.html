RON is a simple config format which looks similar to Rust syntax.

Â§Features

  • Data types
    • Structs, typename optional
    • Tuples
    • Enums
    • Lists
    • Maps
    • Units (())
    • Optionals
    • Primitives: booleans, numbers, string, char
  • Allows nested layout (similar to JSON)
  • Supports comments
  • Trailing commas
  • Pretty serialization
